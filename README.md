# Flutter Coding Standards Skill

[![skills.sh](https://skills.sh/b/brenobattaglin/skills)](https://skills.sh/brenobattaglin/skills)

Comprehensive Flutter and Dart coding standards for AI agents. This skill teaches your AI coding assistant to follow production-quality patterns when generating, reviewing, or refactoring Flutter code.

## What's Covered

| Category | Topics |
|---|---|
| **Project Structure** | Feature-first organization, file naming, directory layout |
| **Naming Conventions** | Classes, variables, booleans, suffixes, file names |
| **State Management** | Bloc/Cubit patterns, immutability, state scoping |
| **Widget Composition** | Composition over inheritance, const constructors, keys |
| **Error Handling** | Result types, exception hierarchy, error display |
| **Testing** | Test structure, naming, unit/widget/golden tests |
| **Performance** | Build optimization, list performance, async patterns |

## Install

```bash
npx skills add brenobattaglin/skills
```
